## üöÄ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16+)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/izwanGit/beruang-landing.git
   ```
2. Navigate to the project directory:
   ```bash
   cd beruang-landing
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
